# 🛍️ E-Commerce Product Explorer & Cart Management App

A modern **React + Vite based E-Commerce Frontend Application** that simulates the core shopping experience of an online store.

Users can:

- Browse products
- Search items instantly
- Filter by category
- Sort by price
- View product details
- Add products to cart
- Manage cart items
- View total summary

This project was built as a **college project** to demonstrate practical frontend development using **React fundamentals + modern architecture**.
